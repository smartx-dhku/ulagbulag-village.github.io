## Local installation

```bash
# Install Ruby
#sudo apt install ruby-full

# Install bundler
#gem install bundler

# Install the Ruby Gem
bundle install

# Initialize search data (creates search-data.json)
bundle exec just-the-docs rake search:init

# Run you local Jekyll server
bundle exec jekyll serve
```
